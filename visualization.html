Tools in Visualization

Data Visualization

  1. Strand NGS: An integrated platform that provides analysis, management and visualization tools for next-generation sequencing data.

  2. CIRCOS: A software package for visualizing NGS data and information.

  3. IGV: A high-performance visualization tool for interactive exploration of large, integrated next-generation sequencing data.

  4. Tablet : A lightweight, high-performance graphical viewer for next generation sequence assemblies and alignments.

  5. Bamview : A free interactive display of read alignments in BAM data files. It has been developed by the Pathogen Group at the Sanger Institute.

  6. EagleView : An information-rich genome assembler viewer with data integration capability.

  7. NGSView : A generally applicable, flexible and extensible next-generation sequence alignment editor.

  8. ZOOM Lite : A next-generation sequencing data mapping and visualization software.

Genome Browsers/Editors

  1. UCSC Genome Browser: A genome browser for any genomic region of specific interest.

  2. Genplay : An analyzer and browser for high-throughput data that is being developed by the Stem Cell Genomic Unit at the Albert Einstein College of Medicine of Yeshiva University in New York City.

  3. Savant : A next-generation genome browser designed for the latest generation of genome data.

  4. ABrowse : A customizable genome browser framework .

  5. Integrated Genomic Browser : An easy-to-use, highly customizable genome browser you can use to view and explore genomic data and annotations, especially RNA-Seq and ChIP-Seq data sets

  6. Artemis : A free genome browser and annotation tool that allows visualisation of sequence features, next generation data and the results of analyses within the context of the sequence, and also its six-frame translation.
